Словарик терминов 3D

  • Точка

    Point, vertex, вершина

    Точка является минимальным элементом любой 3D-модели. Точка является бесконечно малым объектом (у точки нет объема), для описания точки достаточно трех чисел - ее координат по осям X, Y, Z. Геометрией объекта называют совокупность всех его точек, их относительное расположение друг от друга (или можно сказать так: точки описывают геометрию объекта).
